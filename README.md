# 🚀 Marz Node UI

پنل سبک و ساده برای مدیریت نودهای Marzneshin و تنظیمات Xray از طریق SSH.  
این UI با **FastAPI + Uvicorn** نوشته شده و روی پورت `9000` اجرا می‌شود.

---

## ✨ قابلیت‌ها

- ✅ مدیریت نودها از طریق SSH
- ✅ ویرایش و نمایش JSON مستقیماً داخل UI
- ✅ اجرای خودکار `docker compose up -d / down` بعد از اعمال تغییرات
- ✅ Setup Wizard برای تنظیم اولیه
- ✅ ذخیره تنظیمات در فایل `data/config.json`
- ✅ اجرای سرویس با systemd
- ✅ نصب و حذف تک‌دستوری
🆕 قابلیت‌های جدید

💾 Backup Node: گرفتن بکاپ کامل از نود (کل پوشه‌ی /opt/marznode/<label>) و ذخیره خودکار در data/backups

♻️ Restore Backup: برگردوندن آخرین بکاپ نود با یک کلیک + ریستارت اتوماتیک داکر (اینباندها و تنظیمات دقیقاً مثل قبل برمی‌گردن)

🧠 تغییر هسته / ورژن Xray: اجرای خودکار اسکریپت آپدیت هسته و سوییچ ورژن Xray از داخل UI بدون SSH دستی به زودی آپدیت بعدی🙏

🛡️ Soft Delete نودها: دیسبل کردن نود بدون پاک شدن Host و Inbound ها (امن و بدون ریسک)


✨ نتیجه: همه کارهای حیاتی نودها (بکاپ، ریستور، آپدیت هسته، کنترل و کانفیگ) یک‌جا، سریع و بدون دردسر انجام می‌شه.
